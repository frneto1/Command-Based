# Command-Based #
Código de programação baseada em comandos completo para FIRST Robotics Competition (FRC).  
Inclui programação para:  
-> Tank Drive.  
-> Encoder.  
-> Infravermelho.  
-> Intake.  
-> Lançador (Com PID + Feedfoward).  
-> Limelight.  
-> Alinhamento do robô com a apriltag (Com PID).  
-> Robô de escalada (Com PID).  
-> Trajectory Following.  

Projetado e programado por estudantes do ensino médio, membros da equipe HYDRA FRC #9163.
