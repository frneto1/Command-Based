# Command-Based #
Código de programação baseada em comandos completo para FIRST Robotics Competition (FRC).  
Inclui programação para:  
:point_right: Locomoção para Tank Drive.  
:point_right: Encoder.  
:point_right: Infravermelho.  
:point_right: Intake.  
:point_right: Lançador (Com PID + Feedfoward).  
:point_right: Limelight.  
:point_right: Alinhamento do robô com a apriltag (Com PID).  
:point_right: Robô de escalada (Com PID).  
:point_right: Trajectory Following.  
:point_right: Simulação de arena com o Autodesk Synthesis/Fission.

Projetado e programado por estudantes do ensino médio, membros da equipe HYDRA FRC #9163.
